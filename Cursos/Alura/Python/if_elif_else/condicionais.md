# Condicionais if, elif e else  

condicionais são estruturas de código que permite executar diversos blocos de código dependendo se a condição é falsa ou verdadeira  
sempre vem acompanhado de dois pontos  
o else não é obrigatório  
o elif nos permite criar mais uma condição antes de cair no else  

## Conjunto de Operadores de Comparação  
>, <, ==, !=, <=, =>

## Operadores Lógicos 
ao usarmos operadores condicionais é normal utilizarmos operadores lógicos para combinar mais de uma condição, os mais usados são o:  
AND: retornar verdadeiro se todas as condições forem verdadeiras
OR: retorna verdadeiro se pelo menos uma das condições for verdadeira
